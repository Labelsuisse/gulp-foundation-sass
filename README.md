## Installation

```sh
$ git clone git@github.com:Labelsuisse/gulp-starter-kit.git ./

$ npm install

```

## Config paths
in gulp-settings.js file

```javascript
const settings = {
    "src": "src_assets", // source files path
    "dest": "webroot" // destination files path
}
```

## Run commands

### Initialize files

```sh
$ gulp init
```

### Wath files
```sh
$ gulp default
```


### Minify files
```sh
$ gulp minify
```
